Welcome! This is a simple calculator page. Here's how to use it:

  • Input two values on the textboxes. The left one is for the first value while the right one is for the second.
  • The value order matters when using division and subtraction.
    • 5 / 10 ≠ 5
    • 5 - 10 ≠ 5
  • After inputting the desired values in the corresponding textboxes, select an operator.
    • ' + ' : Addition
    • ' - ' : Subtraction
    • ' * ' : Multiplication
    • ' / ' : Division
    • ' % ' : Remainder

Input two values.


Select and click on an operator.
